ECSE848L Internship Program-II

Course Type-Core L-T-P Format 10-30-1 Credits-5

We will be doing our Internship Training Program in Crowe Horwath. We will learn NetSuite Functional and NetSuite Implementation.

Our task we be mainly customising the requirement of clients from USA and Middle East.

COURSE-SPECIFIC LEARNING OUTCOMES

CO1:Theory and Implementation of NetSuite.

CO2: It will help us understanding how clients requirements are fulfilled, how their business are handled using NetSuite.

DETAILED SYLLABUS

Module 1 (W1, W2, W3, W4)

What are Client Scripts in NetSuite? Creating a Client Script, Uploading the Client Script, Creating a Script Record in NetSuite, Deploying the Client Script, Testing the Client Script, Client Script – Entry Points, pageInit, fieldChanged, validateField, postSourcing, lineInit, validateLine, validateInsert, validateDelete, Difference Between validateDelete, validateInsert and validateLine, sublistChanged, Difference Between sublistChanged and Other Entry Points, saveRecord, Difference Between saveRecord and Other Entry Points. Modules in User Event Script, currentRecord Module.

Module 2 (W5, W6, W7, W8)

Introduction to User Event Script, Lifecycle Contexts in User Event Scripts, Entry Point in User Event Script, beforeLoad(context), beforeSubmit(context), afterSubmit (context), context Object, Creating User Event Script, Modules in User Event Script, record Module, url Module.

Module 3 (W9, W10, W11, W12)

Introduction to Scheduled Script, Where is a Scheduled Script Used?, How is it different from other scripts? Creating a Scheduled Script, Entry Point of a Scheduled Script. Governance in Scheduled Script. Modules in Scheduled Script, search Module.

Module 4 (W13, W14, W15, W16)

Introduction to Map Reduce Script, Where is a Map Reduce Script Used? Creating a Map Reduce Script, Entry Point of a Scheduled Script. context Object, Governance in Map Reduce Script. Modules in Map Reduce Script, query Module.

Module 5 (W17, W18, W19, W20)

Introduction to Suitelet Script, Where is a Suitelet Script Used? Creating a Suitelet Script, Entry Point of a Suitelet Script, context Object, Calling a Suitelet Script, scriptlet.nl, Frontend Suitelet vs Backend Suitelet, Modules in Suitelet Script, http Module, url Module, serverWidget Module.

Module 6 (W21, W22, W23, W24)

Introduction to RESTlet Script, Where is a RESTlet Script Used? Creating a RESTlet Script, Entry Point of a RESTlet Script, context Object, Calling a RESTlet Script, Modules in RESTlet Script.

CROWE HORWATH – BILLABLE AND NON BILLABLE PROJECTS

Module 1 (W1, W2, W3, W4, W5, W6)

NCC-Learnings, Pinnacle(US Client)-Scheduled Script: Automated Inactive of Sales Order.

Module 2 (W7, W8, W9, W10, W11, W12)

NCC-Learnings, Pinnacle(US Client)- Client Script and Suitelet: Automate Sublist Values.

Module 3 (W13, W14, W15, W16, W17, W18)

Module 4 (W19, W20, W21, W22, W23, W24, W25)
